页面布局

通栏布局：

浏览器左侧固定，右侧自适应。
1.flex布局，左侧固定宽，右侧flex：1
2.两个盒子分别左浮右浮，右侧盒子减去左侧盒子的宽 100% - 宽
3.左侧盒子左浮动，右侧盒子设置 margin-left：左侧盒子的宽

圣杯布局：

左右两侧固定，中间自适应。
1.使用浮动和定位实现。
2.左右盒子加float分别左浮，右浮，中间盒子百分比减去俩盒子的宽。
3.flex布局，左右给固定宽，中间给flex：1 。

双飞翼布局：

和圣杯布局相似，但是元素结构有点变化，没用到定位。
